Python Keylogger — Automated Keystroke Capture & Email Delivery
~/projects/keylogger/README.md
A Python-based keylogger built for educational purposes to demonstrate real-world keystroke logging behavior. It silently captures every keystroke—including special keys and formatting—and automatically exfiltrates detailed logs to a designated email address at custom intervals using the SMTP protocol.
  • Comprehensive Keystroke Capture — Utilizes the `pynput` library to accurately record all alphanumeric and special keys globally.
  • Automated Exfiltration — Features an integrated SMTP client that packages and silently emails logs to an external server at predetermined intervals.
  • Background Execution — Designed to run stealthily in the background as a demonstration of persistent threat behavior.
  • Threaded Architecture — Uses Python's `threading` module to ensure the logging loop runs continuously without blocking the email dispatch timer.

MailGuard — Email Forensic & Authentication Scanner
~/projects/mailguard-email-security
A rule-based email threat detection utility designed to identify spam, phishing attempts, and fraudulent relays. Built with a Flask API backend and vanilla HTML/JS interface, it parses headers deterministically and scores threats against customizable sensitivity profiles.
  • Rule-Based Analyzer — Deterministic threat scanner utilizing local keyword arrays to flag phishing urgency, scam triggers, and spam patterns.
  • Security Header Audit — Extracts hop timelines, client IPs, and routing paths from raw mail headers, verifying domain and routing states.
  • File Attachment Scanner — Inspects incoming email attachment list extensions, flagging dangerous binary executables, scripts, and archives.
  • TLD & URL Shortener Audits — Validates email body links against blacklisted top-level domains and redirects from URL shortening services.
  • Sensitivity Control Settings — Manages thresholds, whitelist, and blacklist domains, saving parameters directly to local config settings.
  • Threat Telemetry Dashboard — Returns mock telemetry stats mapping false positive rates, top attack categories, and global geographic relay origins.
  • Flask Backend API — Python-based backend service exposing CORS-enabled endpoint routes serving both analysis metrics and configuration data.
  • Responsive Security Console — Simple web dashboard displaying header detail summaries, classification tables, and threat logs.
